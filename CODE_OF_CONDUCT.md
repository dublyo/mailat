# Code of Conduct

## Our Standard

This project is a professional space for respectful collaboration.

Participants are expected to:
- Be respectful and constructive.
- Focus feedback on code, behavior, and outcomes.
- Assume good intent and ask clarifying questions.
- Avoid harassment, insults, and discriminatory language.

## Unacceptable Behavior

Examples include:
- Personal attacks, harassment, or hate speech.
- Trolling, intimidation, or repeated disruptive behavior.
- Sharing private information without permission.
- Any conduct that makes collaboration unsafe.

## Enforcement

Maintainers may edit, remove, or reject comments, issues, or pull requests that violate this policy.
Repeated or severe violations may result in temporary or permanent bans from project spaces.

## Reporting

Report conduct issues privately to:
- `conduct@mailat.co`

Include links/screenshots and brief context so maintainers can investigate.

## Scope

This policy applies in all project spaces, including issues, pull requests, discussions, and chat channels used for this project.
